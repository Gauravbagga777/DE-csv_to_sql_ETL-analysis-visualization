# DE-csv_to_sql_ETL,Analysis_&_Visualization 
This is an ETL pipeline made on jupyter notebook,python and MySQL.

It extracts data from multiple csv files and loads it in the MySQL Database with the help of multiple modules like pandas,os,csv.
It includes complex SQL queries with multiple joins and CTE's for pivoting and analysing data
of 5000 Employees. It also uses matplotlib for Data visualization 

The data files for the project are taken from Mockaroo.com
